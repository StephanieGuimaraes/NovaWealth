# NovaWealth
NovaWealth is a modern financial education platform delivering smart analysis, up-to-date market news, and deep insights into global finance and digital innovation. Alongside financial content for investors, we also create technical resources for developers working in fintech, blockchain, and data-driven financial systems.

## Resumo da arquitetura

Frontend: Next.js (App Router) + React Compiler, Tailwind CSS, fontes Inter/Inter Tight

Backend: Strapi (headless CMS) — usado para criar/editar artigos, categorias, tags, autores e servir imagens via API

Deploy: Vercel (Next) + Railway / Render / Strapi Cloud (Strapi) — vou incluir notas de deploy.
